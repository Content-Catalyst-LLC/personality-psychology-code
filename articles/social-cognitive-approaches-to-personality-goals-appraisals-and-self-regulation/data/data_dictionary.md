# Data Dictionary

Dataset: `synthetic_social_cognitive_personality.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `occasion` | Repeated-measures occasion number |
| `situation_type` | Synthetic situation class |
| `goal_activation` | Strength of active goal pursuit in the occasion |
| `threat_appraisal` | Degree to which the situation is appraised as threat |
| `challenge_appraisal` | Degree to which the situation is appraised as challenge or opportunity |
| `self_efficacy` | Belief in capacity to act effectively in the situation |
| `self_regulation` | Capacity for monitoring, inhibition, planning, and effort regulation |
| `emotional_arousal` | Synthetic emotional activation in the occasion |
| `perceived_support` | Perceived relational or environmental support |
| `prosocial_behavior` | Synthetic prosocial or constructive response |
| `avoidance_behavior` | Synthetic withdrawal, avoidance, or disengagement response |
| `task_persistence` | Synthetic persistence toward task or goal completion |
| `appraisal_balance` | Challenge appraisal minus threat appraisal |
| `regulation_capacity` | Composite of self-efficacy and self-regulation |
| `approach_orientation` | Composite of goal activation, challenge appraisal, and self-efficacy |
| `avoidance_pressure` | Composite of threat appraisal, emotional arousal, and low support |

The dataset is synthetic and not representative of any actual person, classroom, workplace, clinical population, family, or institution.
