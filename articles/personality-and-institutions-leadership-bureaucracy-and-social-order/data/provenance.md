# Data Provenance

The dataset in this directory is synthetic and generated by the upgrade script. It is not derived from human participants, personnel records, survey respondents, administrative data, leadership assessments, workplace analytics, clinical records, or any real institution.

## Why synthetic data?

Synthetic data allow researchers, students, and readers to reproduce the code examples without using sensitive personal information or implying claims about actual organizations.

## What the data can support

- Code demonstration
- Reproducibility testing
- Workflow validation
- Conceptual examples of institutional-personality modeling
- Demonstrations of group summaries, role-fit metrics, and interaction logic

## What the data cannot support

- Claims about real employees, leaders, agencies, organizations, governments, or institutions
- Personnel decisions
- Hiring or promotion decisions
- Psychological assessment
- Workplace surveillance
- Diagnostic interpretation
- Ranking institutions or individuals
