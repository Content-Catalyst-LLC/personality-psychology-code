# Data Dictionary

Dataset: `synthetic_personality_institutions_bureaucracy.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible institutional-personality structure but do not describe real individuals, real workers, real leaders, real agencies, or actual organizations.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic participant / officeholder identifier |
| `institutional_unit` | string | Synthetic organizational or institutional unit |
| `role_type` | string | Synthetic role category |
| `conscientiousness` | numeric | Synthetic trait score for reliability, order, and self-regulation |
| `agreeableness` | numeric | Synthetic trait score for cooperation, civility, and interpersonal orientation |
| `emotional_stability` | numeric | Synthetic score for calm, consistency, and low volatility |
| `openness` | numeric | Synthetic score for adaptive judgment and openness to complexity |
| `dark_trait_pressure` | numeric | Synthetic index of antagonistic, exploitative, or domination-oriented pressure |
| `bureaucratic_fit` | numeric | Synthetic measure of fit between person and institutional role |
| `discretion_level` | numeric | Synthetic measure of decision latitude |
| `accountability_strength` | numeric | Synthetic measure of oversight, transparency, and role constraint |
| `leadership_rating` | numeric | Synthetic outcome representing perceived leadership conduct |
| `institutional_performance` | numeric | Synthetic outcome representing unit or role performance |
| `institutional_trust` | numeric | Synthetic outcome representing trust or legitimacy |
| `role_clarity` | numeric | Synthetic measure of how clearly obligations are defined |

## Important caution

The dataset is not representative of any real institution or population. It should be used only to test code and illustrate analysis patterns.
