# Data Dictionary

Dataset: `synthetic_personality_work_leadership.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible work-personality structure but do not describe real employees, leaders, teams, organizations, or actual job performance.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic worker / officeholder identifier |
| `role_family` | string | Synthetic occupational role family |
| `organizational_context` | string | Synthetic organizational culture / context label |
| `extraversion` | numeric | Synthetic Big Five extraversion score |
| `agreeableness` | numeric | Synthetic Big Five agreeableness score |
| `conscientiousness` | numeric | Synthetic Big Five conscientiousness score |
| `neuroticism` | numeric | Synthetic Big Five neuroticism score |
| `openness` | numeric | Synthetic Big Five openness score |
| `emotional_stability` | numeric | Synthetic emotional stability score |
| `dark_trait_pressure` | numeric | Synthetic antagonistic / manipulative / derailment-pressure index |
| `role_fit` | numeric | Synthetic match between person profile and role demands |
| `accountability` | numeric | Synthetic measure of oversight, feedback, and institutional constraint |
| `job_performance` | numeric | Synthetic job-performance outcome |
| `leadership_emergence` | numeric | Synthetic leadership visibility / emergence outcome |
| `leadership_effectiveness` | numeric | Synthetic leadership effectiveness / stewardship outcome |
| `counterproductive_work_behavior` | numeric | Synthetic counterproductive workplace behavior index |
| `teamwork_quality` | numeric | Synthetic teamwork and cooperation outcome |
| `burnout_risk` | numeric | Synthetic burnout-risk outcome |

## Important caution

The dataset is not representative of any actual workplace or population. It should be used only to test code and illustrate analysis patterns.
