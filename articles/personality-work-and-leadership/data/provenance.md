# Data Provenance

The dataset in this directory is synthetic and generated by the upgrade script. It is not derived from employees, managers, personnel records, leadership assessments, workplace analytics, human-resources files, performance reviews, surveys, clinical records, or any real organization.

## Why synthetic data?

Synthetic data allow researchers, students, and readers to reproduce the code examples without using sensitive personnel information or implying claims about actual employees, leaders, teams, or institutions.

## What the data can support

- Code demonstration
- Reproducibility testing
- Workflow validation
- Conceptual examples of work-personality modeling
- Demonstrations of performance, leadership emergence, effectiveness, and derailment-risk analysis

## What the data cannot support

- Claims about real workers or organizations
- Hiring, promotion, or termination decisions
- Workplace surveillance
- Psychological assessment
- Personnel ranking
- Leadership selection
- Claims about actual performance or institutional quality
