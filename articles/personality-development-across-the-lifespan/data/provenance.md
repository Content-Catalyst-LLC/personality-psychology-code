# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from surveys, intervention studies, schools, workplaces, clinical records, developmental assessments, cohort panels, therapy records, app data, wearable data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, families, cultures, cohorts, developmental pathways, institutions, or public systems.
