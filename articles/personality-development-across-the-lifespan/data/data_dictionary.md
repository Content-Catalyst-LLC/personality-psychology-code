# Data Dictionary

Dataset: `synthetic_personality_lifespan.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `wave` | Measurement wave label |
| `wave_numeric` | Numeric wave value used for longitudinal models |
| `age` | Synthetic age at measurement |
| `life_stage` | Synthetic life-stage category |
| `cohort` | Synthetic historical cohort label |
| `cultural_context` | Synthetic cultural-context label |
| `neuroticism` | Synthetic negative emotionality / emotional instability trait score |
| `extraversion` | Synthetic sociability, energy, and approach tendency score |
| `conscientiousness` | Synthetic reliability, discipline, and follow-through score |
| `openness` | Synthetic curiosity, imagination, and receptivity to complexity score |
| `agreeableness` | Synthetic cooperation, compassion, and interpersonal concern score |
| `role_investment` | Synthetic investment in role structures that may scaffold development |
| `state_practice_frequency` | Synthetic repeated trait-relevant state practice |
| `perceived_support` | Synthetic social or environmental support for development |

The dataset is synthetic and not representative of actual children, adolescents, adults, older adults, families, schools, workplaces, cultures, cohorts, health systems, or institutions.
