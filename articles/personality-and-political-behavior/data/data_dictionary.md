# Data Dictionary

Dataset: `synthetic_personality_political_behavior.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible political-personality structure but do not describe real voters, real citizens, real political parties, real countries, or actual political outcomes.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic respondent identifier |
| `country_context` | string | Synthetic political-context label |
| `political_system_type` | string | Synthetic political-system category |
| `extraversion` | numeric | Synthetic Big Five extraversion score |
| `agreeableness` | numeric | Synthetic Big Five agreeableness score |
| `conscientiousness` | numeric | Synthetic Big Five conscientiousness score |
| `neuroticism` | numeric | Synthetic Big Five neuroticism score |
| `openness` | numeric | Synthetic Big Five openness score |
| `political_interest` | numeric | Synthetic political interest score |
| `political_efficacy` | numeric | Synthetic political efficacy score |
| `group_identity_strength` | numeric | Synthetic strength of political/social group attachment |
| `perceived_threat` | numeric | Synthetic perceived political/social threat score |
| `media_exposure` | numeric | Synthetic exposure to political media/cues |
| `civic_opportunity` | numeric | Synthetic civic opportunity / access measure |
| `ideology_score` | numeric | Synthetic ideological-orientation score |
| `political_participation` | numeric | Synthetic political participation score |
| `affective_polarization` | numeric | Synthetic out-group hostility / affective division score |
| `trust_in_institutions` | numeric | Synthetic institutional trust score |
| `leadership_authority_preference` | numeric | Synthetic preference for strong/authority-oriented leadership |

## Important caution

The dataset is not representative of any actual political system or population. It should be used only to test code and illustrate analysis patterns.
