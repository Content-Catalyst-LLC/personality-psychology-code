# Data Provenance

The dataset in this directory is synthetic and generated by the upgrade script. It is not derived from voters, surveys, political files, campaign data, polling data, social media data, private records, administrative records, or any real population.

## Why synthetic data?

Synthetic data allow researchers, students, and readers to reproduce the code examples without using sensitive political information or implying claims about actual voters, groups, countries, parties, or institutions.

## What the data can support

- Code demonstration
- Reproducibility testing
- Workflow validation
- Conceptual examples of personality-politics modeling
- Demonstrations of participation, ideology, identity-threat, and institutional-trust pathways

## What the data cannot support

- Claims about real voters or political groups
- Claims about national character
- Campaign targeting
- Psychological profiling
- Political persuasion systems
- Voter scoring or ranking
- Ideological diagnosis
- Claims about actual party coalitions or electoral outcomes
