# Data Dictionary

Dataset: `synthetic_personality_culture_universality.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible statistical structure but do not describe real individuals or real populations.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic participant identifier |
| `culture_group` | string | Synthetic cultural sample label |
| `openness` | numeric | Synthetic trait score for openness |
| `conscientiousness` | numeric | Synthetic trait score for conscientiousness |
| `extraversion` | numeric | Synthetic trait score for extraversion |
| `agreeableness` | numeric | Synthetic trait score for agreeableness |
| `neuroticism` | numeric | Synthetic trait score for neuroticism |
| `honesty_humility` | numeric | Synthetic trait score for HEXACO-style honesty-humility |
| `context_collectivism` | numeric | Synthetic contextual index representing relational / collectivist context |
| `behavioral_manifestation` | numeric | Synthetic behavioral outcome influenced by trait and context |
| `survey_language_family` | string | Synthetic language-family label for methodological examples |

## Important caution

The data are not representative of any real society, culture, language group, or population. They should be used only to test code and illustrate analysis patterns.
