# Data Provenance

The dataset in this directory is synthetic and generated by the upgrade script. It is not derived from human participants, survey respondents, private records, clinical data, or any real cross-cultural dataset.

## Why synthetic data?

Synthetic data allow researchers, students, and readers to reproduce the code examples without using sensitive personal information or making claims about real populations.

## What the data can support

- Code demonstration
- Reproducibility testing
- Workflow validation
- Conceptual examples of cross-cultural trait modeling
- Demonstrations of group summaries and matrix similarity

## What the data cannot support

- Claims about real cultures
- Claims about national character
- Psychological assessment
- Clinical interpretation
- Hiring or workplace screening
- Ranking populations
- Policy claims about personality distributions
