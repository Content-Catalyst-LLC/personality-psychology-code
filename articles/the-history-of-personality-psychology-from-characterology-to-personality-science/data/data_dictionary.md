# Data Dictionary

Dataset: `synthetic_personality_history_items.csv`

| Column | Description |
|---|---|
| `participant_id` | Synthetic participant identifier |
| `item1`–`item60` | Synthetic broad personality item pool used for dimensionality inspection |
| `c1`–`c6` | Synthetic conscientiousness item set |
| `extraversion_score` | Synthetic broad extraversion score |
| `agreeableness_score` | Synthetic broad agreeableness score |
| `conscientiousness_score` | Synthetic broad conscientiousness score |
| `neuroticism_score` | Synthetic broad neuroticism score |
| `openness_score` | Synthetic broad openness score |
| `conscientiousness_t1` | Synthetic time-1 conscientiousness score |
| `conscientiousness_t2` | Synthetic time-2 conscientiousness score |
| `behavior_score` | Synthetic behavior criterion used for person-situation modeling |
| `trait_score` | Synthetic trait predictor used for person-situation modeling |
| `situation_strength` | Synthetic situation-strength predictor |
| `person_situation_interaction` | Synthetic interaction term |
| `characterology_typology_index` | Synthetic index representing residual typological thinking |
| `psychometric_structure_index` | Synthetic index representing trait-structure / measurement maturity |
| `person_situation_index` | Synthetic index representing person-situation integration |
| `narrative_identity_index` | Synthetic index representing narrative and biographical integration |
| `measurement_invariance_caution_index` | Synthetic index representing cultural / measurement-equivalence caution |
| `historical_method_maturity_index` | Synthetic index representing the transition toward modern personality science |

Dataset: `synthetic_personality_history_scores_for_sql.csv`

A compact score-only version for SQL examples.

The datasets are synthetic and not representative of real respondents, clients, patients, students, workers, communities, cultures, organizations, languages, institutions, or historical populations.
