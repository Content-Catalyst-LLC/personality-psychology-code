# Data Dictionary

Dataset: `synthetic_personality_dynamics_data.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `assessment_context` | Synthetic context for the repeated-measures scenario |
| `occasion` | Repeated-measure occasion index |
| `trait_extraversion` | Person-level extraversion trait standing |
| `trait_conscientiousness` | Person-level conscientiousness trait standing |
| `trait_neuroticism` | Person-level neuroticism / negative emotionality trait standing |
| `state_extraversion` | Momentary extraverted state expression |
| `state_conscientiousness` | Momentary conscientious state expression |
| `state_neuroticism` | Momentary negative-emotionality state expression |
| `situation_valence` | Positivity or negativity of the momentary situation |
| `situation_sociality` | Degree to which the situation is socially engaging |
| `situation_demand` | Degree of task or role demand |
| `situation_evaluation` | Degree of evaluation, scrutiny, or performance pressure |
| `positive_affect` | Momentary positive affect |
| `negative_affect` | Momentary negative affect |
| `goal_pressure` | Momentary pressure to meet a goal, standard, or demand |
| `autonomy_support` | Momentary support for agency, choice, and volitional action |
| `state_inertia_marker` | Synthetic marker of carryover from prior state patterns |
| `dynamic_signature_score` | Synthetic summary of how strongly state expression follows person-specific situation patterns |

The dataset is synthetic and not representative of actual people, clients, students, workers, teams, organizations, schools, workplaces, cultures, or institutions.
