# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from experience-sampling studies, clinical records, workplace records, school records, employee data, student data, mobile sensing data, social media data, app data, observer reports, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, organizations, teams, schools, workplaces, communities, cultures, or institutions.
