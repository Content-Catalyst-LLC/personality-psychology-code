# Data Dictionary

Dataset: `synthetic_personality_measurement_data.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `assessment_context` | Synthetic assessment setting or use context |
| `s1`–`s5` | Self-report conscientiousness items on a 1–7 scale |
| `o1`–`o5` | Observer-report conscientiousness items on a 1–7 scale |
| `attention_check` | Synthetic response-quality flag coded 1 for passed and 0 for failed |
| `careless_response_risk` | Synthetic estimate of careless or low-effort responding |
| `social_desirability_pressure` | Synthetic estimate of pressure to present favorably |
| `self_missing_count` | Count of missing self-report items |
| `observer_missing_count` | Count of missing observer-report items |
| `self_conscientiousness` | Mean of self-report items |
| `observer_conscientiousness` | Mean of observer-report items |
| `self_other_discrepancy` | Self-report score minus observer-report score |
| `absolute_self_other_discrepancy` | Absolute self–other difference |
| `method_effect_index` | Synthetic index representing reporting-source and context effects |
| `reliability_context_score` | Synthetic score representing the quality of the measurement context |
| `professional_reflection_score` | Synthetic score representing usefulness for low-stakes professional education or methodological reflection |

The dataset is synthetic and not representative of actual people, clients, students, workers, teams, organizations, schools, workplaces, cultures, or institutions.
