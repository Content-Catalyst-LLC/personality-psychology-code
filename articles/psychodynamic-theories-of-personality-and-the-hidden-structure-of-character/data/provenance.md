# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from therapy notes, clinical records, diagnostic interviews, psychological assessments, treatment programs, patient surveys, family records, institutional data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive clinical or human-subject data and avoiding claims about actual people, patients, therapists, families, clinics, diagnoses, or institutions.
