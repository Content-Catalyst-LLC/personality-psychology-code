# Data Dictionary

Dataset: `synthetic_lexical_descriptors.csv`

| Column | Description |
|---|---|
| `respondent_id` | Synthetic respondent/target identifier |
| `adj1`–`adj100` | Synthetic lexical descriptor ratings |
| `sociable_cluster_score` | Synthetic score for an extraversion-like descriptor cluster |
| `reliable_cluster_score` | Synthetic score for a conscientiousness-like descriptor cluster |
| `compassionate_cluster_score` | Synthetic score for an agreeableness-like descriptor cluster |
| `anxious_cluster_score` | Synthetic score for a neuroticism/emotionality-like descriptor cluster |
| `imaginative_cluster_score` | Synthetic score for an openness/intellect-like descriptor cluster |
| `social_reliability_outcome` | Synthetic outcome linked to reliability descriptors |
| `interpersonal_trust_outcome` | Synthetic outcome linked to compassion and low antagonism descriptors |
| `expressive_engagement_outcome` | Synthetic outcome linked to sociability and imagination descriptors |
| `lexical_visibility_index` | Synthetic marker of how socially visible a descriptor profile is |
| `lexical_abundance_index` | Synthetic marker of lexical richness or descriptor density |
| `structural_centrality_index` | Synthetic marker of centrality in recovered trait structure |
| `descriptor_redundancy_index` | Synthetic marker of overlap among near-synonymous descriptors |
| `cross_language_caution_index` | Synthetic marker for language/translation sensitivity |

Dataset: `synthetic_lexical_scores_for_sql.csv`

A compact score-only version for SQL examples.

The datasets are synthetic and not representative of real respondents, clients, patients, students, workers, communities, cultures, languages, organizations, or institutions.
