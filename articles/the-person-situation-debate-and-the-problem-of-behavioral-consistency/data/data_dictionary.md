# Data Dictionary

Dataset: `synthetic_person_situation_data.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `assessment_context` | Synthetic context for the repeated-measures scenario |
| `occasion` | Repeated-measure occasion index |
| `trait_score` | Broad synthetic person-level trait standing |
| `trait_extraversion` | Person-level extraversion standing |
| `trait_conscientiousness` | Person-level conscientiousness standing |
| `trait_neuroticism` | Person-level neuroticism / negative emotionality standing |
| `state_extraversion` | Momentary extraverted state expression |
| `state_conscientiousness` | Momentary conscientious state expression |
| `state_assertiveness` | Momentary assertive behavior/state expression |
| `state_withdrawal` | Momentary withdrawal or guardedness |
| `situation_demand` | Task, role, or responsibility demand |
| `situation_sociality` | Social opportunity or social engagement |
| `situation_evaluation` | Evaluation, scrutiny, or performance pressure |
| `situation_trust` | Interpersonal trust or safety in the situation |
| `situation_autonomy` | Freedom, control, and agency in the situation |
| `situation_threat` | Threat, risk, punishment, or insecurity |
| `behavioral_consistency_marker` | Synthetic marker of aggregated consistency across situations |
| `conditional_signature_score` | Synthetic marker of if–then pattern strength |
| `state_inertia_marker` | Synthetic marker of carryover from prior state patterns |

The dataset is synthetic and not representative of actual people, clients, students, workers, teams, organizations, schools, workplaces, cultures, or institutions.
