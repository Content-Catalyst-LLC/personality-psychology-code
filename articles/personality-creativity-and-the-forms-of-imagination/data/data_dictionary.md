# Data Dictionary

File: `synthetic_personality_creativity.csv`

| Variable | Type | Description |
|---|---:|---|
| `id` | integer | Synthetic participant identifier. |
| `openness` | numeric | Openness-related disposition score. |
| `intellect` | numeric | Abstract/intellectual engagement score. |
| `conscientiousness` | numeric | Order, reliability, persistence-adjacent trait score. |
| `extraversion` | numeric | Sociability, expressiveness, and activation score. |
| `agreeableness` | numeric | Cooperation, warmth, and prosocial orientation score. |
| `neuroticism` | numeric | Negative emotionality score. |
| `persistence` | numeric | Execution, follow-through, and sustained-effort score. |
| `social_support` | numeric | Perceived support/opportunity context score. |
| `domain` | categorical | Synthetic primary creative context: arts, science, design, or everyday. |
| `divergent_thinking` | numeric | Ideational generation outcome. |
| `creative_achievement` | numeric | Realized-output / achievement outcome. |
| `everyday_creativity` | numeric | Adaptive everyday creativity outcome. |

All numeric variables use a synthetic 0-100 scale. Higher values indicate more of the construct.
