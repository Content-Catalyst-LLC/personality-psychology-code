# Data Dictionary

Dataset: `synthetic_personality_creativity.csv`

All rows are synthetic demonstration records.

| Variable | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic participant identifier. |
| `domain` | categorical | Illustrative creative domain: `artistic`, `scientific`, `technical`, or `everyday`. |
| `openness` | numeric | Synthetic openness-to-experience score, scaled 1–7. |
| `intellect` | numeric | Synthetic intellect/abstract-engagement score, scaled 1–7. |
| `conscientiousness` | numeric | Synthetic conscientiousness score, scaled 1–7. |
| `extraversion` | numeric | Synthetic extraversion score, scaled 1–7. |
| `agreeableness` | numeric | Synthetic agreeableness score, scaled 1–7. |
| `neuroticism` | numeric | Synthetic neuroticism score, scaled 1–7. |
| `persistence` | numeric | Synthetic persistence/execution-capacity score, scaled 1–7. |
| `social_support` | numeric | Synthetic support/opportunity score, scaled 1–7. |
| `divergent_thinking` | numeric | Synthetic divergent-thinking outcome, scaled 0–100. |
| `creative_achievement` | numeric | Synthetic creative-achievement outcome, scaled 0–100. |
| `everyday_creativity` | numeric | Synthetic everyday-creativity outcome, scaled 0–100. |

## Measurement note

The variables are deliberately simplified. They are intended to support transparent computational examples, not measurement claims about real individuals.
