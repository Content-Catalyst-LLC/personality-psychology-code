# Data Dictionary

Dataset: `synthetic_traits_character_morality.csv`

| Column | Description |
|---|---|
| `participant_id` | Synthetic participant ID |
| `evaluation_context` | Synthetic social or institutional evaluation context |
| `honesty_humility` | Sincerity, fairness, modesty, and resistance to exploitation |
| `conscientiousness` | Discipline, reliability, responsibility, and follow-through |
| `agreeableness` | Cooperation, compassion, warmth, and interpersonal concern |
| `emotional_stability` | Emotional steadiness and regulation under pressure |
| `openness` | Imagination, reflection, and receptivity to complexity |
| `moral_identity` | Centrality of moral values to self-understanding |
| `practical_judgment` | Ability to apply moral understanding to concrete circumstances |
| `institutional_accountability` | Synthetic accountability, transparency, and constraint in the setting |
| `power_pressure` | Synthetic pressure associated with hierarchy, advantage, or authority |
| `social_desirability_pressure` | Synthetic pressure to appear morally good or socially approved |
| `ethical_behavior` | Synthetic morally relevant behavior score |
| `integrity_rating` | Synthetic integrity evaluation |
| `trustworthiness_rating` | Synthetic trustworthiness evaluation |
| `descriptive_trait_reliability` | Composite of Honesty-Humility, conscientiousness, and agreeableness |
| `moral_character_index` | Composite of ethical behavior, integrity, and trustworthiness |
| `judgment_context_index` | Composite of practical judgment and institutional accountability |
| `trait_character_gap` | Moral character index minus descriptive trait reliability |

The dataset is synthetic and not representative of any actual person, workplace, community, institution, or moral character profile.
