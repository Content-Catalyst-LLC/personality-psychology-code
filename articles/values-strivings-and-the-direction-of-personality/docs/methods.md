# Methods Notes

This scaffold models values, strivings, and personality direction through linked dimensions:

1. basic value priorities
2. value composites and value-tension indices
3. personal striving orientations
4. autonomy, competence, relatedness, and ownership
5. striving conflict
6. motivational quality
7. life-direction coherence
8. life satisfaction
9. directional profiles and vulnerable motivational patterns

The workflow is designed for transparent method demonstration. A real study would require validated value measures, careful cultural adaptation, sampling design, measurement-invariance testing, goal-coding protocols, attention to power and institutional context, missing-data planning, ethical review, and safeguards against misuse.
