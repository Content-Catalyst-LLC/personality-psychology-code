# Data Dictionary

Dataset: `synthetic_values_strivings_direction.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `value_context` | Synthetic context shaping value formation, ownership, and striving conflict |
| `benevolence` | Synthetic value priority for close care, helpfulness, and concern for known others |
| `universalism` | Synthetic value priority for broad moral concern, justice, and wider human or ecological care |
| `self_direction` | Synthetic value priority for autonomy, independent thought, and self-directed action |
| `achievement` | Synthetic value priority for competence, accomplishment, and socially recognized success |
| `power` | Synthetic value priority for status, control, influence, or dominance |
| `security` | Synthetic value priority for safety, order, and stability |
| `tradition` | Synthetic value priority for inherited practices, continuity, or communal/religious custom |
| `stimulation` | Synthetic value priority for novelty, challenge, and exploration |
| `striving_meaning` | Degree to which strivings are organized around meaningful purpose |
| `striving_status` | Degree to which strivings are organized around status, recognition, or superiority |
| `striving_care` | Degree to which strivings are organized around care, responsibility, and relationship |
| `striving_autonomy` | Degree of volition and self-endorsement in strivings |
| `striving_competence` | Felt effectiveness in pursuing strivings |
| `striving_relatedness` | Relational support and belonging around strivings |
| `striving_conflict` | Degree of conflict among central values and personal strivings |
| `striving_ownership` | Degree to which strivings are experienced as owned rather than merely imposed |
| `life_satisfaction` | Synthetic life-satisfaction indicator |
| `self_transcendence` | Mean of benevolence and universalism |
| `self_enhancement` | Mean of achievement and power |
| `openness_to_change` | Mean of self-direction and stimulation |
| `conservation` | Mean of security and tradition |
| `value_tension_self_transcendence_enhancement` | Absolute distance between self-transcendence and self-enhancement |
| `value_tension_openness_conservation` | Absolute distance between openness to change and conservation |
| `value_tension_total` | Mean value-tension score |
| `striving_prosocial_orientation` | Mean of striving meaning and striving care |
| `motivational_quality` | Mean of autonomy, competence, relatedness, and ownership |
| `life_direction_coherence` | Synthetic composite linking motivational quality, prosocial orientation, self-transcendence, and conflict |

The dataset is synthetic and not representative of actual people, communities, workplaces, schools, cultures, religious traditions, political groups, or institutions.
