# Data Dictionary

Dataset: `synthetic_maladaptive_personality_structure.csv`

| Column | Description |
|---|---|
| `participant_id` | Synthetic participant ID |
| `clinical_context` | Synthetic context label |
| `negative_affectivity` | Maladaptive emotional reactivity / distress tendency |
| `detachment` | Maladaptive withdrawal / restricted intimacy tendency |
| `antagonism` | Maladaptive hostility / entitlement / low empathy tendency |
| `disinhibition` | Maladaptive impulsivity / poor constraint tendency |
| `psychoticism` | Maladaptive eccentricity / cognitive-perceptual dysregulation tendency |
| `anankastia` | Maladaptive rigidity / perfectionism / compulsive control |
| `identity_impairment` | Impairment in identity coherence |
| `self_direction_impairment` | Impairment in goals, agency, and self-direction |
| `empathy_impairment` | Impairment in understanding others as separate persons |
| `intimacy_impairment` | Impairment in mutuality, closeness, and stable relationship |
| `self_functioning_impairment` | Composite of identity and self-direction impairment |
| `interpersonal_functioning_impairment` | Composite of empathy and intimacy impairment |
| `functioning_impairment` | Overall personality-functioning impairment |
| `maladaptive_trait_burden` | Mean maladaptive trait burden |
| `severity_trait_interaction` | Interaction between functioning impairment and trait burden |
| `rigidity` | Inflexibility of response pattern |
| `pervasiveness` | Cross-context recurrence of the pattern |
| `contextual_stress` | Synthetic contextual stress burden |
| `perceived_support` | Synthetic perceived support |
| `clinical_severity` | Synthetic clinical severity score |
| `clinical_liability` | Synthetic dimensional liability score |
| `threshold_zone_indicator` | Synthetic indicator of proximity to threshold-zone concern |

The dataset is synthetic and not representative of any actual clinical population.
