# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from personality assessments, workplace records, school records, clinical records, employee data, student data, surveys, app data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, organizations, teams, schools, workplaces, communities, cultures, or institutions.
