# Data Dictionary

Dataset: `synthetic_personality_relationships_social_functioning.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible relational-personality structure but do not describe real individuals, couples, families, friendships, communities, or actual relationship outcomes.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic person identifier |
| `social_context` | string | Synthetic social context label |
| `relationship_domain` | string | Synthetic relational domain |
| `extraversion` | numeric | Synthetic Big Five extraversion score |
| `agreeableness` | numeric | Synthetic Big Five agreeableness score |
| `conscientiousness` | numeric | Synthetic Big Five conscientiousness score |
| `neuroticism` | numeric | Synthetic Big Five neuroticism score |
| `openness` | numeric | Synthetic Big Five openness score |
| `empathy` | numeric | Synthetic empathy / perspective-taking score |
| `self_regulation` | numeric | Synthetic self-regulation score |
| `attachment_security` | numeric | Synthetic attachment security score |
| `perceived_support` | numeric | Synthetic perceived support score |
| `relationship_satisfaction` | numeric | Synthetic relationship satisfaction outcome |
| `social_functioning` | numeric | Synthetic broader social functioning outcome |
| `loneliness` | numeric | Synthetic loneliness / isolation outcome |
| `conflict_frequency` | numeric | Synthetic conflict frequency outcome |
| `reciprocity_quality` | numeric | Synthetic reciprocity quality outcome |
| `reputation_trust` | numeric | Synthetic reputation / trust outcome |

## Important caution

The dataset is not representative of any actual person, dyad, family, community, or population. It should be used only to test code and illustrate analysis patterns.
