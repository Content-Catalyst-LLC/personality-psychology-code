# Methods Notes

This scaffold models the psychometric issues around MBTI-style typological classification through linked dimensions:

1. continuous latent preference dimensions
2. observed scores with measurement fluctuation
3. dichotomous threshold cuts
4. four-letter type-code construction
5. boundary proximity and threshold fragility
6. retest type stability
7. type-frequency summaries
8. within-type variation
9. continuous-vs-categorical model comparison
10. information-loss summaries

The workflow is designed for transparent method demonstration. A real study would require validated instruments, access to item-level or scale-level data, reliability estimation, construct validation, measurement-invariance work, prediction targets defined in advance, and strict limits on applied use.
