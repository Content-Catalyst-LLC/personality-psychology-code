# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from MBTI assessments, workplace records, school records, counseling records, employee data, survey panels, app data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, organizations, teams, communities, cultures, schools, workplaces, or institutions.
