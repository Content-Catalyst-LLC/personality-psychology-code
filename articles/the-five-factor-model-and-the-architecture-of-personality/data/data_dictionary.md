# Data Dictionary

Dataset: `synthetic_ffm_items.csv`

| Column | Description |
|---|---|
| `respondent_id` | Synthetic respondent identifier |
| `item1`–`item60` | Synthetic broad Five-Factor personality item pool |
| `c1`–`c6` | Synthetic conscientiousness domain items |
| `o1`–`o3` | Synthetic orderliness facet items |
| `i1`–`i3` | Synthetic industriousness facet items |
| `extraversion_score` | Synthetic broad extraversion score |
| `agreeableness_score` | Synthetic broad agreeableness score |
| `conscientiousness_score` | Synthetic broad conscientiousness score |
| `neuroticism_score` | Synthetic broad neuroticism score |
| `openness_score` | Synthetic broad openness score |
| `sociability_score` | Synthetic extraversion facet score |
| `assertiveness_score` | Synthetic extraversion facet score |
| `compassion_score` | Synthetic agreeableness facet score |
| `politeness_score` | Synthetic agreeableness facet score |
| `orderliness_score` | Synthetic conscientiousness facet score |
| `industriousness_score` | Synthetic conscientiousness facet score |
| `anxiety_score` | Synthetic neuroticism facet score |
| `volatility_score` | Synthetic neuroticism facet score |
| `aesthetics_score` | Synthetic openness facet score |
| `intellect_score` | Synthetic openness facet score |
| `outcome_score` | Synthetic narrow criterion used for domain/facet comparison |
| `broad_life_functioning` | Synthetic broad outcome used for bandwidth comparison |
| `relationship_functioning` | Synthetic interpersonal outcome linked to agreeableness and extraversion |
| `creative_engagement` | Synthetic openness-linked outcome |
| `emotional_distress` | Synthetic neuroticism-linked outcome |
| `facet_profile_dispersion` | Synthetic marker of within-domain facet unevenness |
| `hierarchy_consistency_index` | Synthetic marker of item-facet-domain coherence |
| `bandwidth_fidelity_gap` | Synthetic marker comparing broad and narrow predictive alignment |
| `domain_facet_alignment` | Synthetic marker of agreement between broad domain and facet summaries |

Dataset: `synthetic_ffm_scores_for_sql.csv`

A compact score-only version for SQL examples.

The datasets are synthetic and not representative of real respondents, clients, patients, students, workers, teams, cultures, organizations, or institutions.
