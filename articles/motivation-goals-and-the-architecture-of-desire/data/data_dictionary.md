# Data Dictionary

Dataset: `synthetic_motivation_goals_desire.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `motivation_context` | Synthetic context shaping need support, goal ownership, and goal conflict |
| `autonomy_goal` | Synthetic priority for self-direction, agency, and choice |
| `achievement_goal` | Synthetic priority for accomplishment, competence display, and mastery |
| `belonging_goal` | Synthetic priority for connection, affiliation, and social inclusion |
| `security_goal` | Synthetic priority for safety, stability, threat reduction, and protection |
| `meaning_goal` | Synthetic priority for purpose, identity significance, and personally meaningful direction |
| `status_goal` | Synthetic priority for recognition, status, rank, visibility, or comparative success |
| `goal_conflict` | Degree of conflict among central goals |
| `goal_ownership` | Degree to which goals are experienced as owned and self-endorsed |
| `autonomy_support` | Environmental support for volition and self-direction |
| `competence_support` | Environmental support for effectiveness and skill development |
| `relatedness_support` | Environmental support for connection and belonging |
| `conscientiousness` | Synthetic trait-relevant self-regulatory tendency |
| `persistence_score` | Synthetic persistence indicator |
| `adaptive_disengagement` | Capacity to revise or release goals when they are harmful, impossible, or misaligned |
| `well_being` | Synthetic wellbeing indicator |
| `total_goal_intensity` | Sum of core goal priorities |
| `approach_orientation` | Mean of autonomy, achievement, belonging, and meaning goals |
| `avoidance_security_orientation` | Security-goal priority |
| `status_orientation` | Status-goal priority |
| `need_support` | Mean of autonomy, competence, and relatedness support |
| `motivational_quality` | Mean of goal ownership, need support, and meaning goal |
| `life_direction_coherence` | Synthetic composite linking motivational quality, approach orientation, goal ownership, and goal conflict |

The dataset is synthetic and not representative of actual people, patients, students, workers, families, communities, institutions, workplaces, schools, cultures, religious traditions, or political groups.
