# Data Dictionary

Dataset: `synthetic_personality_wellbeing_mental_health.csv`

This is a synthetic dataset for reproducible educational examples. Values are generated to resemble plausible personality-wellbeing structure but do not describe real people, patients, clinical samples, communities, or actual mental-health outcomes.

| Column | Type | Description |
|---|---:|---|
| `participant_id` | string | Synthetic person identifier |
| `age_band` | string | Synthetic life-stage category |
| `life_context` | string | Synthetic social / structural context label |
| `extraversion` | numeric | Synthetic Big Five extraversion score |
| `agreeableness` | numeric | Synthetic Big Five agreeableness score |
| `conscientiousness` | numeric | Synthetic Big Five conscientiousness score |
| `neuroticism` | numeric | Synthetic Big Five neuroticism score |
| `openness` | numeric | Synthetic Big Five openness score |
| `coping_effectiveness` | numeric | Synthetic coping effectiveness score |
| `perceived_support` | numeric | Synthetic perceived social support score |
| `stress_burden` | numeric | Synthetic chronic stress burden score |
| `positive_affect` | numeric | Synthetic positive affect score |
| `negative_affect` | numeric | Synthetic negative affect score |
| `life_satisfaction` | numeric | Synthetic life satisfaction score |
| `meaning_purpose` | numeric | Synthetic meaning and purpose score |
| `wellbeing_score` | numeric | Synthetic broad wellbeing score |
| `distress_score` | numeric | Synthetic distress score |
| `flourishing_score` | numeric | Synthetic flourishing / eudaimonic functioning score |
| `social_functioning` | numeric | Synthetic social functioning score |
| `treatment_access` | numeric | Synthetic access to mental-health or supportive care resources |
| `sleep_quality` | numeric | Synthetic sleep quality score |

## Important caution

The dataset is not representative of any actual person, clinical group, social group, country, community, or health system. It should be used only to test code and illustrate analysis patterns.
