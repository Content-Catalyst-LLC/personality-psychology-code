# Data Provenance

The dataset in this directory is synthetic and generated by the upgrade script. It is not derived from clinical records, therapy records, patient surveys, wearable data, school records, workplace records, social media data, diagnostic interviews, or any real population.

## Why synthetic data?

Synthetic data allow researchers, students, and readers to reproduce the code examples without using sensitive mental-health information or implying claims about actual persons, patients, communities, diagnoses, or care systems.

## What the data can support

- Code demonstration
- Reproducibility testing
- Workflow validation
- Conceptual examples of personality-wellbeing modeling
- Demonstrations of distress, flourishing, coping, support, and two-continua profile analysis

## What the data cannot support

- Clinical diagnosis
- Treatment decisions
- Mental-health scoring
- Personality assessment
- Risk prediction
- Population ranking
- Claims about actual distress, flourishing, or diagnosis
- Claims about real communities or care systems
