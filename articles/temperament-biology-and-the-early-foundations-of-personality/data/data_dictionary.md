# Data Dictionary

Dataset: `synthetic_temperament_personality_longitudinal.csv`

| Column | Description |
|---|---|
| `child_id` | Synthetic child identifier |
| `developmental_context` | Synthetic developmental context |
| `inhibition_t1` | Early behavioral inhibition / caution in novelty |
| `negative_affect_t1` | Early negative affectivity, frustration, distress, or emotional reactivity |
| `surgency_t1` | Early approach, activity, positive anticipation, and social energy |
| `effortful_control_t1` | Early attentional and behavioral self-regulation |
| `parenting_support_t1` | Caregiver sensitivity, support, scaffolding, and co-regulation |
| `family_stress_t1` | Family stress, instability, or chronic strain |
| `classroom_support_t2` | Later school/classroom support and developmental scaffolding |
| `peer_support_t2` | Later peer acceptance, relational support, or social belonging |
| `institutional_stability_t2` | Stability and responsiveness of school/community institutions |
| `conscientiousness_t2` | Later conscientiousness-related outcome |
| `neuroticism_t2` | Later neuroticism / negative emotionality outcome |
| `social_confidence_t2` | Later social confidence and adaptive approach |
| `regulation_skill_t2` | Later self-regulation and emotional recovery skill |
| `reactivity_regulation_balance` | Synthetic summary of early reactivity relative to regulation |
| `environmental_support_index` | Synthetic summary of caregiving, classroom, peer, and institutional support |
| `developmental_risk_index` | Synthetic risk marker combining reactivity, stress, and weak support |
| `adaptive_pathway_score` | Synthetic adaptive pathway marker under support and regulation |

The dataset is synthetic and contains no real child, family, clinical, educational, health, or demographic data.
