# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from child records, school records, clinical records, developmental assessments, parent reports, teacher reports, surveys, app data, genetic data, health records, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive child, family, clinical, educational, or human-subject data and avoiding claims about actual people, families, schools, workplaces, communities, cultures, or institutions.
