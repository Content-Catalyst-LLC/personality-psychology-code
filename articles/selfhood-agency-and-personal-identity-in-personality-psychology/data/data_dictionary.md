# Data Dictionary

Dataset: `synthetic_selfhood_agency_identity.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `identity_context` | Synthetic context shaping agency, recognition, and self-continuity |
| `past_self` | Synthetic representation of felt connection to the past self |
| `present_self` | Synthetic representation of present self-clarity |
| `future_self` | Synthetic representation of felt connection to the future self |
| `intentional_clarity` | Clarity about aims, motives, and reasons for action |
| `action_ownership` | Felt ownership or endorsement of action |
| `self_efficacy` | Felt capacity to act effectively |
| `external_constraint` | Synthetic external constraint, coercion, or blocked opportunity |
| `social_recognition` | Degree of social recognition, affirmation, and narrative legitimacy |
| `value_commitment_gap` | Distance between endorsed values and enacted commitments |
| `identity_integration` | Synthetic index of coherent identity organization |
| `well_being` | Synthetic wellbeing indicator |
| `past_present_gap` | Absolute distance between past and present self-representations |
| `present_future_gap` | Absolute distance between present and future self-representations |
| `temporal_self_continuity` | Average of past-present and present-future continuity |
| `agency_index` | Composite of intentional clarity, action ownership, self-efficacy, and inverse constraint |
| `situated_agency_index` | Agency index extended by social recognition |
| `identity_alignment` | Inverse of value–commitment gap |

The dataset is synthetic and not representative of actual people, patients, students, workers, families, communities, cultures, legal subjects, or institutions.
