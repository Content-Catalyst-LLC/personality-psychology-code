# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from surveys, intervention studies, therapy records, clinical records, school records, workplace records, coaching programs, app data, wearable data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, clinical groups, students, workers, institutions, or intervention programs.
