# Data Dictionary

Dataset: `synthetic_personality_change_intervention.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `wave` | Measurement wave label |
| `wave_numeric` | Numeric wave value used for longitudinal models |
| `intervention_group` | Synthetic intervention or comparison group |
| `age` | Synthetic age at measurement |
| `neuroticism` | Synthetic negative emotionality / emotional instability trait score |
| `extraversion` | Synthetic sociability, energy, and approach tendency score |
| `conscientiousness` | Synthetic reliability, discipline, and follow-through score |
| `openness` | Synthetic curiosity, imagination, and receptivity to complexity score |
| `agreeableness` | Synthetic cooperation, compassion, and interpersonal concern score |
| `role_investment` | Synthetic investment in a role that may scaffold personality change |
| `state_practice_frequency` | Synthetic frequency of repeated trait-relevant state practice |
| `perceived_support` | Synthetic social or environmental support for change |

The dataset is synthetic and not representative of actual people, patients, students, workers, therapy clients, coaching participants, schools, workplaces, health systems, or institutions.
