# Data Dictionary

Dataset: `synthetic_personality_twin_data.csv`

| Column | Description |
|---|---|
| `pair_id` | Synthetic twin-pair identifier |
| `zygosity` | Synthetic twin type: `MZ` or `DZ` |
| `genetic_relatedness` | Simplified relatedness value: 1.0 for MZ and 0.5 for DZ |
| `twin1_trait` | Synthetic personality trait score for twin 1 |
| `twin2_trait` | Synthetic personality trait score for twin 2 |
| `twin1_temperament_reactivity` | Synthetic temperament/reactivity score for twin 1 |
| `twin2_temperament_reactivity` | Synthetic temperament/reactivity score for twin 2 |
| `twin1_effortful_control` | Synthetic effortful-control score for twin 1 |
| `twin2_effortful_control` | Synthetic effortful-control score for twin 2 |
| `family_stress` | Shared family stress exposure |
| `social_support` | Shared and contextual support availability |
| `socioeconomic_security` | Synthetic socioeconomic stability/security indicator |
| `educational_stability` | Synthetic stability of schooling/institutional environment |
| `shared_environment_index` | Summary of pair-shared environmental support/stress conditions |
| `nonshared_environment_index` | Pair-level magnitude of nonshared environmental difference |
| `trait_mean` | Mean of both twins' trait scores |
| `trait_difference` | Absolute difference between twins' trait scores |
| `gxe_marker` | Synthetic marker representing gene–environment moderation signal |
| `rge_marker` | Synthetic marker representing gene–environment correlation signal |
| `developmental_context_score` | Synthetic score summarizing supportive developmental conditions |

The dataset is synthetic and not representative of actual twins, families, patients, students, workers, communities, cultures, or institutions. It contains no real genetic, clinical, educational, employment, insurance, or health data.
