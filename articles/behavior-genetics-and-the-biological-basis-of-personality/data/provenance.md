# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from twin registries, genomic data, clinical records, workplace records, school records, family records, genetic tests, consumer DNA services, surveys, app data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, families, genetic groups, organizations, schools, workplaces, cultures, or institutions.
