# Data Dictionary

Dataset: `synthetic_self_concept_self_esteem_self_knowledge.csv`

| Column | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `self_system_context` | Synthetic context shaping self-concept, recognition, devaluation, and self-knowledge |
| `self_warmth` | Self-rating of warmth |
| `self_conscientiousness` | Self-rating of conscientiousness |
| `self_emotional_stability` | Self-rating of emotional stability |
| `self_openness` | Self-rating of openness |
| `other_warmth` | Synthetic observer rating of warmth |
| `other_conscientiousness` | Synthetic observer rating of conscientiousness |
| `other_emotional_stability` | Synthetic observer rating of emotional stability |
| `other_openness` | Synthetic observer rating of openness |
| `actual_self` | Synthetic actual-self rating |
| `ideal_self` | Synthetic ideal-self rating |
| `ought_self` | Synthetic ought-self rating |
| `self_esteem` | Synthetic self-esteem indicator |
| `social_recognition` | Degree of social recognition, affirmation, and narrative legitimacy |
| `external_devaluation` | Degree of external devaluation, misrecognition, or degrading treatment |
| `well_being` | Synthetic wellbeing indicator |
| `warmth_gap` | Absolute self–other difference for warmth |
| `conscientiousness_gap` | Absolute self–other difference for conscientiousness |
| `emotional_stability_gap` | Absolute self–other difference for emotional stability |
| `openness_gap` | Absolute self–other difference for openness |
| `self_other_gap_mean` | Mean self–other gap across domains |
| `self_knowledge_accuracy` | Inverse self–other gap scaled to 0–1 |
| `actual_ideal_discrepancy` | Distance between actual self and ideal self |
| `actual_ought_discrepancy` | Distance between actual self and ought self |
| `total_self_discrepancy` | Mean of actual–ideal and actual–ought discrepancies |
| `self_concept_positivity` | Mean of self-rated warmth, conscientiousness, emotional stability, and openness |

The dataset is synthetic and not representative of actual people, patients, students, workers, families, communities, schools, workplaces, cultures, institutions, or clinical populations.
