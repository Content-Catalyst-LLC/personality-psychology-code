# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from surveys, interviews, clinical records, school records, workplace data, therapy notes, legal records, app data, social media data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, families, communities, schools, workplaces, cultures, institutions, clinical populations, or public systems.
