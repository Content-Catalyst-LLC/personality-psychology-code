# Data Provenance

The dataset is generated synthetically by this repository script. It is not derived from personality inventories, clinical records, employment assessments, school records, human-subject research, app data, survey panels, psychometric vendors, organizational data, or any real population.

Synthetic data are used to make the workflow reproducible while avoiding sensitive human-subject data and avoiding claims about actual people, workplaces, schools, communities, cultures, or institutions.
