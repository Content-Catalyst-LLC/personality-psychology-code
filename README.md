# Personality Psychology Code

Open computational scaffolding for the Personality Psychology knowledge series.

This repository supports article-level examples for traits, trait structure, psychometrics, temperament, behavior genetics, person-situation interaction, personality dynamics, assessment, personality types, MBTI critique, motivation, values, selfhood, narrative identity, lifespan personality development, personality change, psychodynamic theory, social-cognitive personality, character, dark traits, maladaptive personality, personality disorders, wellbeing, health, relationships, leadership, institutions, political behavior, culture, and creativity.

## Structure

Each article folder contains:

- `python/`
- `r/`
- `julia/`
- `cpp/`
- `fortran/`
- `c/`
- `rust/`
- `go/`
- `sql/`
- `notebooks/`
- `docs/`
- `data/raw/`
- `data/processed/`
- `outputs/`
